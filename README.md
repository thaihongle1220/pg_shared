# Personal Growth & Self-Understanding System

A personal project for tracking psychological patterns, processing experiences, and building self-awareness.

**Created:** October 2025
**Owner:** Thai, 25 tuổi, HCMC Vietnam

---

## Dành cho Bác sĩ / Therapist

Project này là hệ thống tự theo dõi tâm lý cá nhân, sử dụng AI (Claude) như một công cụ hỗ trợ reflection. Nội dung bao gồm:

- Suy nghĩ thô, cảm xúc, patterns nhận ra
- Sessions thảo luận sâu về trauma, relationships, identity
- Mục tiêu và định hướng cuộc sống

**Để hiểu nhanh nhất, đọc theo thứ tự:**

1. `STATE.md` → Snapshot hiện tại
2. `IDENTITY.md` → Con người, values
3. `daily_notes/25_12_31_session1.txt`
4. `daily_notes/25_12_31_session2.txt`
5. `daily_notes/25_12_31_session3.txt`

## Cách sử dụng

### Nếu muốn hiểu con người:
→ `IDENTITY.md` và `PRINCIPLES.md`

### Nếu muốn hiểu trạng thái hiện tại:
→ `STATE.md` và `GOALS.md`

### Nếu muốn hiểu psychological depth:
→ Các file trong `daily_notes/`, đặc biệt sessions ngày 31/12/2025

### Nếu muốn thấy patterns theo thời gian:
→ `daily_notes/` từ cũ đến mới (25_10_30 → 25_12_31)

4. **Relationship readiness** - genuine criteria hay self-protection?
5. **Trauma integration** - cognitive transcendence vs. somatic processing
